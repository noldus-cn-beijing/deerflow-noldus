# 常见行为学指标外分析要求
行为学指标很容易理解。速度、距离、进入次数，EthoVision里面的那么多因变量，都是行为学指标。

那什么叫做“指标外”的分析要求？

顾名思义，就是不是指标，会被要求分析给导师看的内容。常见的有以下几种：

## 时间分段分析
把整个实验记录按照时间轴线，分成多个 段落 / Time Bin 的要求。一般用来体现行为指标随着实验进程的变化趋势。通常，这样的数据不需要单独设置后导出。常见的有：
- 中间点。二分为前半，后半。
- 三等分为前中后阶段。
- 五等分。
- 取最开始的20%时间和最后的20%时间。

## 分区分析
把整个实验的数据按照分区规划。比如移动距离这个指标，动物在中心区走了多远？在外周区走了多远？在四个象限各又走了多远？要分析这一点，普通的`Raw`是没办法获取分区信息的，**需要让用户在EthoVision里面的数据选择配置-结果中勾选`按分析区输出`后，才能在`Raw`中体现**。

## 事件/状态分段分析
> - “我想要看明期和暗期活动性的区别。”
> - “我想知道亮灯后到电击开始前鼠的移动距离”
> - “我需要知道鼠的活跃度大于10的时间和段落次数”

类似这样的，以实验阶段，硬件状态，特定阈值，为分类条件的分割方法。在传统的EthoVision实验中，需要用户使用`可用区间`，`多条件`，`JavaScript自定义因变量`等，辅以`试验控制日志`和`硬件日志`在软件内进行计算。或导出上述指标和文件，在软件外进行独立的比较。而在EthoInsight中，
- 早期可以要求用户在EthoVision软件中设置好对应因变量，导出，在`Raw`中体现，直接读取。
- 最终版需要能够根据`Raw`和`试验控制日志`及`硬件日志`独立计算。
