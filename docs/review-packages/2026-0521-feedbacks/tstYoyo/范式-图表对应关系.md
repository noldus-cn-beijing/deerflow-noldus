# 范式-图表对应关系
这里我们需要明确一下图表的定义。
表好说，结果表格，什么范式都需要。
但是图，会分为两类 —— `可视化图` 和 `结果图`

- 可视化图  
这类图主要表示被试动物位置的时空分布，往往不直接反映结果，而是用于给读者一个对位置时空分布的直观感受。最常见的包括：
  - 轨迹图
  - 热区图
- 结果图  
这类图主要用于展示指标结果的对比。有的直接反映结果，如移动距离或速度的柱状图；有的反映结果二次计算后新指标的对比，如聚类或回归相关的散点图及折线图
  - 折线图
  - 柱状图
  - 散点图
  - 直方图
  - 小提琴图
  - 相关性热图
  - 等其它反映数据指标变化的图

下面我们针对MVP阶段各个涉及范式输出图表进行分类整理。

## 高架十字迷宫
### 可视化图
- **轨迹图**：展示动物在开放臂与封闭臂之间的运动路径，按组/个体/移动速度着色
  - 宽高比固定 4:3
- **热区图**：停留时间密度分布，直观反映动物对开放臂的回避程度
  - 宽高比固定 4:3

### 结果图
- **箱线图**：开放臂时间比例、开放臂进入次数等核心指标的组间分布比较
- **柱状图（均值 ± SEM）**：开放臂时间比例均值的组间比较，配显著性标注
- **区域进入分布图**：开放臂 vs 封闭臂进入次数的分组对比
  - 类似 matplotlib中 `plt.eventplot()` 做出来的效果
### 涉及指标
| 指标 | 含义 | 焦虑方向 |
|------|------|----------|
| open_arm_time_ratio | 开放臂时间比例 | 越低越焦虑 |
| open_arm_time | 开放臂时间（秒） | 越低越焦虑 |
| open_arm_entry_count | 开放臂进入次数 | 越低越焦虑 |
| open_arm_entry_ratio | 开放臂进入比例 | 越低越焦虑 |
| total_entry_count | 总进入次数 | 无方向性（反映整体活动量） |


## 旷场
### 可视化图
- **轨迹图**：展示动物在中心区与边缘区之间的运动路径，按组/个体/移动速度着色
  - 宽高比固定 4:3
- **热区图**：停留时间密度分布，直观反映动物对中心区的回避程度
  - 宽高比固定 4:3

### 结果图
- **箱线图**：中心区时间比例、中心区移动距离比例等核心指标的组间分布比较
- **柱状图（均值 ± SEM）**：中心区时间比例均值的组间比较，配显著性标注
- **中心区进入汇总图**：中心区进入次数的分组对比
- **时间进程图**：若实验较长，以3-5分钟为time bin，画折线图，看运动距离和中央区滞留之间的变化（评估习惯化）

### 涉及指标
| 指标 | 含义 | 焦虑方向 |
|------|------|----------|
| center_time_ratio | 中心区时间比例 | 越低越焦虑 |
| center_time | 中心区时间（秒） | 越低越焦虑 |
| center_distance_ratio | 中心区移动距离比例 | 越低越焦虑 |
| center_distance | 中心区移动距离（cm） | 越低越焦虑 |
| center_entry_count | 中心区进入次数 | 越低越焦虑 |

## 悬尾
### 可视化图
无。悬尾实验中动物被倒挂固定，位置数据无空间意义，不做可视化。

### 结果图
- **箱线图**：不动时间、不动次数、首次不动潜伏期的组间分布比较
- **柱状图（均值 ± SEM）**：各指标均值的组间比较，配显著性标注
- **活动强度图**：整个测试期间的活动强度（activity-像素变化）波形图。
- **分布图**：挣扎 vs 放弃挣扎时间分布对比
  - 类似 matplotlib中 `plt.eventplot()` 做出来的效果

### 涉及指标
| 指标 | 含义 | 焦虑方向 |
|------|------|----------|
| immobility_time | 不动时间（秒） | 抑郁样行为核心指标，越高越抑郁样 |
| immobility_latency | 首次不动潜伏期（秒） | 越短越抑郁样 |
| immobility_bout_count | 不动次数 | 越多越抑郁样 |
