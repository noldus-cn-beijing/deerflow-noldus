# EthoInsight MVP 功能验收清单

> **版本**: v1.0 | **日期**: 2026-04-29 | **基于**: PRD v2 DRAFT | **状态**: 待评审

---

## 使用说明

- 每个 Scenario 为一条独立验收项，测试时标记 ✅ 通过 / ❌ 失败 / ⚠️ 部分通过
- "前置条件" 标注运行该 Scenario 前需准备的数据或状态
- "严重度"：P0 = 阻塞发布，P1 = 发布前必须修复，P2 = 可延后一个迭代

## Epic 2: 分析与计算引擎

### US-2.1 全自动模式统计分析

**用户故事**: 作为实验技术员（全自动模式），我希望一键获得完整的统计分析结果，包括正确的检验方法、效应量和显著性，这样我不需要自己选择统计方法。

| ID | Scenario | Given | When | Then | 严重度 |
|----|----------|-------|------|------|--------|
| 2.1.1 | 正态性检验自动执行 | 系统已加载数据并识别范式，用户选择"全自动模式" | 用户点击"开始分析" | 系统对每组每指标执行 Shapiro-Wilk 正态性检验，记录 W 值和 p 值 | P0 |
| 2.1.2 | 正态分布数据选择参数检验 | Shapiro-Wilk 检验结果 p > 0.05（所有组） | 系统自动选择统计方法 | 系统选择参数检验（两组: independent t-test; 多组: one-way ANOVA） | P0 |
| 2.1.3 | 非正态数据选择非参数检验 | Shapiro-Wilk 检验结果 p ≤ 0.05（任一组） | 系统自动选择统计方法 | 系统选择非参数检验（两组: Mann-Whitney U; 多组: Kruskal-Wallis） | P0 |
| 2.1.4 | 效应量自动计算 | 系统已完成统计检验 | 分析结果生成 | 系统报告效应量：参数检验报告 Cohen's d（两组）或 η²（多组）；非参数检验报告 rank-biserial correlation | P0 |
| 2.1.5 | 完整统计报告 | 系统完成所有统计计算 | 查看分析结果 | 每个指标的统计摘要包含：p 值、效应量及大小判定、置信区间（如适用） | P0 |
| 2.1.6 | 多组比较自动校正 | 数据包含 3 个及以上实验组 | 系统执行统计检验 | 系统自动应用多重比较校正（Bonferroni 或 FDR），报告校正前后的 p 值 | P1 |
