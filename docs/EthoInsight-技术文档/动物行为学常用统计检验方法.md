# 动物行为学常用统计检验方法指南

本文档整理了动物行为学研究中常用的统计检验方法，按**差异比较、回归分析、聚类分析、关联与相关、生存分析、降维与综合分析**六大类别组织，并附有选择流程图和各方法详细说明。

## 一、差异比较类检验选择流程图

### 流程图 1：两组比较

研究问题：两组样本是否存在显著差异？  
│  
├── 数据类型？  
│ ├── 连续型（计量资料）→ 继续  
│ └── 分类/等级型（计数/等级资料）→ 见流程图 3  
│  
├── 数据是否配对/重复测量？  
│ ├── 是（同一动物前后测量、同窝对照等）  
│ │ ├── 是否服从正态分布？(Shapiro-Wilk 检验)  
│ │ ├── 是 → 配对 t 检验 (Paired t-test)  
│ │ └── 否 → Wilcoxon 符号秩检验 (Wilcoxon Signed-Rank Test)  
│ │  
│ └── 否（独立两组）  
│ ├── 是否服从正态分布？(Shapiro-Wilk 检验)  
│ ├── 是 → 方差齐性？(Levene 检验)  
│ │ ├── 是 → 独立样本 t 检验 (Independent t-test)  
│ │ └── 否 → Welch t 检验  
│ └── 否 → Mann-Whitney U 检验 / Wilcoxon 秩和检验  

### 流程图 2：三组及以上比较

研究问题：三组及以上样本是否存在显著差异？  
│  
├── 数据类型？  
│ ├── 连续型 → 继续  
│ └── 分类/等级型 → 见流程图 3  
│  
├── 设计类型？  
│ ├── 完全随机设计（各组独立）  
│ │ ├── 各组是否正态 + 方差齐？  
│ │ ├── 是 → 单因素方差分析 (One-Way ANOVA)  
│ │ │ └── ANOVA 显著 → 事后多重比较  
│ │ │ ├── 各组样本量相等 → Tukey HSD  
│ │ │ ├── 各组样本量不等 → Games-Howell  
│ │ │ └── 与对照组比较 → Dunnett's 检验  
│ │ └── 否 → Kruskal-Wallis H 检验  
│ │ └── 显著 → Dunn's 事后检验（Bonferroni 校正）  
│ ├── 随机区组设计（配伍组设计）  
│ │ ├── 是否正态 + 球形假设？(Mauchly 检验)  
│ │ ├── 是 → 两因素方差分析（无重复）  
│ │ └── 否 → Friedman 检验  
│ │ └── 显著 → Nemenyi 事后检验  
│ └── 重复测量设计（同一动物多次测量）  
│ ├── 是否正态 + 球形假设？(Mauchly 检验)  
│ ├── 是 → 重复测量方差分析 (RM-ANOVA)  
│ │ ├── 球形假设满足 → 用标准 F 值  
│ │ └── 球形假设违反 → Greenhouse-Geisser 或 Huynh-Feldt 校正  
│ │ └── 交互效应显著 → 简单效应分析  
│ └── 否 → 对齐秩重复测量方差分析 (Aligned Rank Transform ANOVA)  
│ 或 Friedman 检验  
│  
├── 是否为多因素（如处理 × 时间 × 基因型）？  
│ ├── 是 → 多因素方差分析 (n-Way ANOVA) / 混合设计 ANOVA  
│ │ 注意：多因素设计中协变量 → 协方差分析 (ANCOVA)  
│ └── 否 → 按上述单因素流程  
│  
└── 样本量极小（n < 5/组）？  
├── 是 → 考虑非参数方法或增加样本量  
└── 否 → 按常规流程  

### 流程图 3：分类/等级数据比较

研究问题：分类数据或等级数据是否存在差异？  
│  
├── 两组比较  
│ ├── 四格表（2×2）  
│ │ ├── 总 n ≥ 40 且所有理论频数 ≥ 5？  
│ │ ├── 是 → Pearson 卡方检验 (Chi-square test)  
│ │ ├── 总 n ≥ 40 但有理论频数 1~5？  
│ │ │ └── 连续性校正卡方检验 (Yates's correction)  
│ │ └── 总 n < 40 或理论频数 < 1？  
│ │ └── Fisher 精确检验 (Fisher's Exact Test)  
│ └── 等级数据（如行为评分）  
│ └── Mann-Whitney U 检验  
│  
├── 三组及以上  
│ ├── R×C 列联表  
│ │ ├── 所有理论频数 ≥ 5？  
│ │ ├── 是 → Pearson 卡方检验  
│ │ └── 否 → Fisher-Freeman-Halton 精确检验  
│ └── 等级数据（如行为严重程度评分）  
│ └── Kruskal-Wallis H 检验  
│  
└── 配对分类数据（同一动物前后分类）  
└── McNemar 检验  

## 二、回归分析选择流程图

研究问题：变量间是否存在预测或因果关系？  
│  
├── 因变量类型？  
│ ├── 连续型  
│ │ ├── 自变量个数？  
│ │ ├── 1 个 → 简单线性回归 (Simple Linear Regression)  
│ │ │ ├── 是否满足线性假设？→ 散点图 + 残差分析  
│ │ │ ├── 不满足 → 考虑非线性回归（多项式、对数、指数等）  
│ │ │ └── 异方差/非正态残差 → 加权最小二乘或变量变换  
│ │ └── 多个 → 多元线性回归 (Multiple Linear Regression)  
│ │ ├── 自变量间是否高度共线？(VIF > 10)  
│ │ ├── 是 → 岭回归 (Ridge) / Lasso 回归 / 主成分回归  
│ │ └── 否 → 逐步回归或全模型分析  
│ │ ├── 是否有层次结构（如窝内嵌套）？  
│ │ └── 是 → 混合效应模型 / 多层线性模型  
│ ├── 二分类  
│ │ └── 逻辑回归 (Logistic Regression)  
│ ├── 计数型  
│ │ └── 泊松回归 (Poisson Regression)  
│ │ ├── 过度离散（方差 >> 均值）？  
│ │ ├── 是 → 负二项回归 (Negative Binomial Regression)  
│ │ └── 零过多（大量零值）？  
│ │ └── 零膨胀泊松/负二项回归 (ZIP/ZINB)  
│ ├── 生存时间型  
│ │ └── 见生存分析部分  
│ └── 有重复测量/纵向数据  
│ └── 广义估计方程 (GEE) / 广义线性混合模型 (GLMM)  
│  
├── 因变量为行为序列/轨迹？  
│ └── 广义可加模型 (GAM) 或 功能性数据分析  
│  
└── 因果推断需求？  
└── 结构方程模型 (SEM) 或 中介/调节效应分析  

## 三、聚类分析选择流程图

研究问题：能否将动物/行为模式/神经元活动进行分组？  
│  
├── 是否知道类别数？  
│ ├── 是（已知分组标准）  
│ │ └── 判别分析 (Discriminant Analysis)  
│ └── 否（探索性分析）  
│ ├── 样本量？  
│ ├── 大样本  
│ │ ├── 数据类型？  
│ │ ├── 连续型 → K-Means 聚类 / 层次聚类  
│ │ ├── 混合类型（连续+分类）→ K-Prototypes / Gower 距离 + 层次聚类  
│ │ └── 行为轨迹 → 隐马尔可夫模型 (HMM) / 时间序列聚类  
│ └── 小样本（n < 50）  
│ └── 层次聚类（凝聚法，推荐 Ward 方法）  
│ └── 或基于模型聚类（如高斯混合模型 GMM）  
│  
├── 行为模式分类需求？  
│ ├── 时空行为数据（EthoVision 轨迹）→ DBSCAN / OPTICS  
│ └── 行为序列数据 → 序列聚类 / HMM  
│  
└── 聚类结果验证  
├── 轮廓系数 (Silhouette Score)  
├── Gap Statistic  
└── 内部验证 + 生物学意义验证  

## 四、关联与相关分析选择流程图

研究问题：两个或多个变量之间是否存在关联？  
│  
├── 两个连续变量  
│ ├── 是否服从双变量正态分布？  
│ ├── 是 → Pearson 相关系数 (r)  
│ └── 否 → Spearman 秩相关 (ρ) 或 Kendall τ 相关系数  
│  
├── 两个等级变量  
│ └── Spearman 秩相关 / Kendall τ / 列联系数  
│  
├── 两个分类变量  
│ ├── 四格表 → Phi 系数 (φ)  
│ ├── R×C 表 → Cramér's V / 列联系数  
│ └── 等级分类 → Gamma 系数 / Somers' D  
│  
├── 一个连续 + 一个分类  
│ ├── 分类为二分类 → 点双列相关  
│ └── 分类为多分类 → Eta (η) 系数 / ANOVA  
│  
└── 多变量间关联  
├── 探索变量间结构 → 主成分分析 (PCA) / 因子分析  
├── 两个变量集间关系 → 典型相关分析 (CCA)  
└── 偏相关（控制混杂） → 偏相关分析 (Partial Correlation)  

## 五、生存分析选择流程图

研究问题：事件发生的时间（如发病时间、死亡时间、到达目标时间）  
│  
├── 数据特点：存在删失数据（未观察到事件发生）  
│  
├── 是否只需描述生存曲线？  
│ └── Kaplan-Meier 生存分析  
│ └── 比较两组或多组 → Log-rank 检验  
│  
├── 是否需分析多因素对生存的影响？  
│ └── Cox 比例风险回归 (Cox Proportional Hazards Model)  
│ ├── 比例风险假设检验 → Schoenfeld 残差检验  
│ ├── 假设不满足 → 含时变协变量的 Cox 模型 / 参数生存模型  
│ └── 竞争风险存在 → 竞争风险模型 (Fine-Gray)  
│  
└── 特定应用（动物行为学）  
├── 逃避潜伏期分析 → Cox 回归 / 加速失效时间模型  
└── 行为转换时间分析 → 生存曲线 + Log-rank  

## 六、降维与综合分析

研究目的：数据降维、变量整合或综合评分  
│  
├── 变量降维（减少变量数）  
│ ├── 连续变量 → 主成分分析 (PCA)  
│ ├── 混合类型 → 多重对应分析 (MCA)  
│ └── 行为指标整合 → 行为综合评分（PCA 得分）  
│  
├── 变量结构探索  
│ └── 探索性因子分析 (EFA) / 确认性因子分析 (CFA)  
│  
├── 多变量组间比较  
│ ├── 两组 → Hotelling T² 检验  
│ └── 多组 → MANOVA（多元方差分析）  
│  
└── 综合分析  
├── 路径分析 → 行为因果链推断  
├── 结构方程模型 (SEM) → 复杂关系建模  
└── 元分析 (Meta-Analysis) → 跨研究效应量整合  

## 附录 A：统计检验方法详细说明

### A.1 参数检验

|   |   |   |   |   |
|---|---|---|---|---|
|方法|全称|用法|适用条件|典型应用场景|
|独立样本 t 检验|Independent samples t-test|比较两组独立样本均数差异|两组独立、正态分布、方差齐|比较实验组和对照组的焦虑水平|
|配对 t 检验|Paired t-test|比较配对/前后测量均数差异|配对设计、差值正态分布|同一批动物给药前后的行为变化|
|Welch t 检验|Welch's t-test|比较两组独立样本均数（方差不齐时）|两组独立、正态、方差不齐|两组样本量悬殊或方差不齐|
|单因素 ANOVA|One-Way ANOVA|比较三组及以上独立样本均数|多组独立、正态、方差齐|多剂量组药物效果比较|
|重复测量 ANOVA|RM-ANOVA|比较同一受试者多次测量的差异|重复测量、正态、球形假设|不同时间点行为的纵向追踪|
|多因素 ANOVA|n-Way ANOVA|分析多个因素及其交互效应|多因素设计、正态、方差齐|处理×性别×时间交互效应|
|ANCOVA|Analysis of Covariance|在控制协变量后比较组间差异|协变量与因变量线性相关|控制体重后比较药物效果|
|MANOVA|Multivariate ANOVA|同时比较多个因变量|多个连续因变量、正态、方差齐|同时分析移动距离和中央停留时间|

### A.2 非参数检验

|   |   |   |   |   |
|---|---|---|---|---|
|方法|全称|用法|适用条件|典型应用场景|
|Mann-Whitney U|Wilcoxon Rank-Sum Test|比较两组独立样本的分布差异|两组独立、不要求正态|非正态行为数据组间比较|
|Wilcoxon 符号秩|Wilcoxon Signed-Rank Test|比较配对样本的分布差异|配对设计、不要求正态|前后测量行为评分比较|
|Kruskal-Wallis H|Kruskal-Wallis Test|比较三组及以上独立样本分布|多组独立、不要求正态|多组非正态数据比较|
|Friedman|Friedman Test|比较配伍组/重复测量分布差异|重复测量、不要求正态|同一动物多条件下行为比较|
|McNemar|McNemar's Test|配对分类数据差异检验|配对二分类数据|同一动物前后行为分类变化|

### A.3 事后比较检验

|   |   |   |
|---|---|---|
|方法|用法|适用条件|
|Tukey HSD|所有组间两两比较|方差齐、样本量相等或相近|
|Bonferroni|控制族错误率的两两比较|比较次数较少时适用|
|Games-Howell|方差不齐时的两两比较|方差不齐、样本量不等|
|Dunnett's|各组与对照组比较|仅与对照组比较|
|Dunn's|非参数事后比较|Kruskal-Wallis 事后检验|
|Nemenyi|Friedman 事后检验|多组重复测量非参数|
|Holm|逐步 Bonferroni|比较次数较多时，比 Bonferroni 更有力|
|Sidak|控制族错误率|独立比较，比 Bonferroni 略有力|

### A.4 卡方检验与精确检验

|   |   |   |
|---|---|---|
|方法|用法|适用条件|
|Pearson 卡方|R×C 列联表关联检验|所有理论频数 ≥ 5|
|Yates 校正卡方|2×2 表关联检验|总 n ≥ 40，有理论频数 1~5|
|Fisher 精确检验|2×2 表精确概率|总 n < 40 或理论频数 < 1|
|Fisher-Freeman-Halton|R×C 表精确概率|有理论频数过小的 R×C 表|

### A.5 相关分析

|   |   |   |   |
|---|---|---|---|
|方法|用法|适用条件|值域|
|Pearson r|线性相关强度|双变量正态、线性关系|[-1, 1]|
|Spearman ρ|单调相关强度|至少有序等级|[-1, 1]|
|Kendall τ|单调相关强度（小样本更稳健）|有序等级、小样本|[-1, 1]|
|偏相关|控制混杂后的相关|需满足线性假设|[-1, 1]|
|Cramér's V|分类变量关联强度|R×C 列联表|[0, 1]|
|Phi 系数|2×2 表关联强度|四格表|[-1, 1]|

### A.6 回归分析

|   |   |   |   |
|---|---|---|---|
|方法|因变量类型|关键假设|典型应用|
|线性回归|连续|线性、正态残差、等方差、无共线|行为分数 vs 剂量|
|逻辑回归|二分类|独立性、无完全分离|行为是否出现 vs 预测变量|
|有序逻辑回归|有序多分类|比例优势假设|行为严重程度分级|
|泊松回归|计数|均值 = 方差|特定行为发生次数|
|负二项回归|过离散计数|方差 > 均值|过度离散的行为计数|
|Cox 回归|生存时间|比例风险假设|逃避潜伏期分析|
|岭回归/Lasso|连续|同线性回归|共线性严重的多维行为数据|
|混合效应模型|连续/分类|随机效应正确设定|窝内嵌套、纵向行为数据|
|GLMM|任意（通过链接函数）|分布和链接函数正确|复杂层次结构的行为数据|
|GEE|任意|工作相关矩阵正确设定|重复测量的群体平均效应|

### A.7 聚类分析

|   |   |   |   |   |
|---|---|---|---|---|
|方法|数据类型|需预设 K|适合场景|典型应用|
|K-Means|连续|是|大样本、球状簇|行为模式分类|
|层次聚类（Ward）|连续|否（树状图判断）|小样本|行为表型聚类|
|K-Prototypes|混合|是|连续+分类混合|动物多属性聚类|
|DBSCAN|任意|否|任意形状簇、有噪声|轨迹热点检测|
|GMM|连续|否（BIC 选择）|椭球形簇|行为状态识别|
|HMM|序列|是|时间序列行为状态转换|行为序列状态识别|

### A.8 生存分析

|   |   |   |
|---|---|---|
|方法|用法|关键假设|
|Kaplan-Meier|估计生存函数|删失非信息性|
|Log-rank 检验|比较两组或多组生存曲线|比例风险|
|Cox 回归|多因素生存分析|比例风险、线性对数风险|
|加速失效时间 (AFT)|参数化生存建模|特定分布假设（Weibull 等）|

## 附录 C：正态性与方差齐性检验

|   |   |   |   |
|---|---|---|---|
|检验方法|用途|样本量建议|备注|
|Shapiro-Wilk|正态性检验|n < 5000|推荐首选|
|Kolmogorov-Smirnov|正态性检验|大样本|对大样本过于敏感|
