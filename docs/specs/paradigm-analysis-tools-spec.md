## 这份文档是做什么的？

我们在做一个自动分析工具：用户上传 EthoVision XT 导出的轨迹数据后，系统自动完成**指标计算 → 统计检验 → 生成图表 → 结果解读**。

为了让系统正确分析每种实验，我们需要你告诉我们：

1. 每种实验**应该算哪些指标**
2. 这些指标在 EthoVision 导出的数据里**对应哪些列**
3. 结果应该用**什么图表**展示
4. 什么样的数值算**正常**，什么算**异常**
5. 组间差异意味着什么（如"焦虑样表型"）

## 可用的图表类型

以下图表我们已经实现了，你可以从中选择每个范式默认生成哪些图：

| 图表名称 | 长什么样 | 适合展示什么 |

| 箱线图 | 经典方框+须线 | 各组的分布和中位数 |
| 小提琴图 | 对称的曲线轮廓 | 数据分布的形状 |
| 柱状图 + 误差线 | 柱子+SEM/SD 误差线 | 各组均值比较 |

| 轨迹图 | 动物运动路径 | 直观展示活动模式 |
| 时间序列折线图 | 指标随时间的变化曲线 | 时间依赖的指标 |
| 相关性热力图 | 颜色矩阵 | 多个指标之间的相关性 |
