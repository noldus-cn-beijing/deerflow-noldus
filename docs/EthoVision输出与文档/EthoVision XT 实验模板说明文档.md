# EthoVision XT 内置实验模板说明文档

> 基于 EthoVision XT 19 Application Manual 和 Help 文档整理

## 1. 旷场实验（Open Field Test）

| 项目 | 说明 |
|------|------|
| **宫体形状** | 圆形（circular）或方形（square） |
| **宫体模板** | Open field (round) / Open field (square) |
| **区域模板** | Border（边缘区）、Quadrants（象限划分）、Center（中央区） |
| **范式类型** | 单动物/多动物追踪 |
| **适用动物** | 大鼠、小鼠、斑马鱼 |
| **实验目的** | 评估动物的自发活动水平和探索行为；量化焦虑样行为（中央区停留时间反映焦虑程度） |
| **适用领域** | 神经精神疾病模型（焦虑症、抑郁症）、药理学研究、遗传学筛选、行为基线评估 |
| **主要参数** | 总运动距离、运动速度、中央区/边缘区停留时间、区域转换次数、排便颗粒计数 |

### 1.1 斑马鱼旷场变体

- **Novel Tank Diving Test**：使用梯形水槽，量化斑马鱼的底部偏好（thigmotaxis/geotaxis），评估焦虑样行为
- **区域划分**：上半区/下半区

## 2. Morris 水迷宫（Morris Water Maze Test）

| 项目 | 说明 |
|------|------|
| **宫体形状** | 圆形水池 |
| **宫体模板** | Circular arena（适用于水迷宫） |
| **区域模板** | Platform + Quadrants（平台 + 象限）；Platform + Quadrants + Corridors + Border（含走廊与边缘） |
| **范式类型** | 单动物追踪 |
| **适用动物** | 大鼠、小鼠 |
| **实验目的** | 评估空间学习与记忆能力；动物需利用空间线索找到隐藏在水下的平台 |
| **适用领域** | 阿尔茨海默病模型、海马损伤研究、神经可塑性、认知老化、药物筛选 |
| **主要参数** | 潜伏期（找到平台时间）、游泳路径长度、游泳速度、象限停留时间、穿越平台次数、Gallagher 接近度评分 |

### 特殊区域

- **Whishaw 走廊（Corridor）**：可选区域模板，用于分析搜索策略（如直线搜索 vs 边缘搜索）

## 3. 新物体识别实验（Novel Object Recognition Test, NOR）

| 项目 | 说明 |
|------|------|
| **宫体形状** | 圆形或方形旷场 |
| **宫体模板** | Open field (round) / Open field (square) |
| **区域模板** | Novel object zones（围绕物体定义区域） |
| **范式类型** | 单动物追踪，鼻尖检测（强制开启） |
| **适用动物** | 大鼠、小鼠 |
| **实验目的** | 评估识别记忆能力；基于动物对新奇物体的偏好探究天然倾向 |
| **适用领域** | 记忆障碍研究、认知增强药物评估、神经退行性疾病模型、发育毒性评估 |
| **主要参数** | 探究时间（新物体 vs 熟悉物体）、辨别指数（Discrimination Index）、到物体距离 |

### 实验阶段

1. **适应期（Habituation）**：空旷场自由探索
2. **训练期（Familiarization）**：放置两个相同物体
3. **测试期（Test）**：一个熟悉物体 + 一个新物体

## 4. 高架十字迷宫（Elevated Plus Maze Test）

| 项目 | 说明 |
|------|------|
| **宫体形状** | "+"字形，两条开臂 + 两条闭臂，高架于地面 |
| **宫体模板** | Elevated Plus Maze |
| **区域模板** | Open arms（开臂）、Closed arms（闭臂）、Center（中央平台） |
| **范式类型** | 单动物追踪 |
| **适用动物** | 大鼠、小鼠 |
| **实验目的** | 评估焦虑样行为；动物在开臂与闭臂之间的时间分配反映焦虑水平 |
| **适用领域** | 抗焦虑药物筛选、焦虑症模型、应激研究、GABA 系统药理学 |
| **主要参数** | 开臂/闭臂停留时间、开臂/闭臂进入次数、开臂时间百分比、总运动距离、头部探究（stretch-attend posture） |

## 5. 社交偏好实验 / 三箱实验（Sociability Cage Test / Three-Chamber Test）

| 项目 | 说明 |
|------|------|
| **宫体形状** | 三箱长方形（中央箱 + 左箱 + 右箱，中间有隔门） |
| **宫体模板** | Sociability cage |
| **区域模板** | 按箱体划分三个区域 |
| **范式类型** | 单动物追踪 |
| **适用动物** | 小鼠 |
| **实验目的** | 评估社交偏好和社会新奇偏好 |
| **适用领域** | 自闭症谱系障碍（ASD）模型、精神分裂症社交缺陷研究、催产素/加压素系统研究 |
| **主要参数** | 各箱体停留时间、与物体/陌生鼠接触时间、社交偏好指数 |

### 实验阶段

1. **适应期**：空三箱自由探索
2. **社交偏好期**：一侧放陌生鼠 + 一侧放空物体笼
3. **社会新奇偏好期**：一侧放熟悉鼠 + 一侧放新陌生鼠

---

## 6. 社交交互实验（Social Interaction Test）

| 项目 | 说明 |
|------|------|
| **宫体形状** | 圆形或方形旷场 |
| **宫体模板** | Open field (round) / Open field (square)，或 PhenoTyper |
| **区域模板** | 基于旷场区域 + 接近区域（proximity zones） |
| **范式类型** | 双动物追踪（需 Social Interaction 模块） |
| **适用动物** | 大鼠、小鼠（标记/未标记均可） |
| **实验目的** | 量化两只动物之间的社交行为（接近、跟随、社交探究等） |
| **适用领域** | 社交行为研究、精神分裂症模型、抗精神病药物评估、攻击行为研究 |
| **主要参数** | 接近次数、接触持续时间、跟随时间、社交距离、两动物间距离 |

---

## 7. 辐射臂迷宫（Radial Arm Maze Test）

| 项目 | 说明 |
|------|------|
| **宫体形状** | 八臂（标准）星形迷宫，从中央平台向外辐射 |
| **宫体模板** | Radial Arm Maze（8-arm） |
| **区域模板** | Central platform + 各臂 |
| **范式类型** | 单动物追踪 |
| **适用动物** | 大鼠、小鼠 |
| **实验目的** | 评估工作记忆和参考记忆；动物需学习在不重复进入已探索臂的情况下找到所有食物奖励 |
| **适用领域** | 前额叶皮层功能研究、海马记忆机制、阿尔茨海默病模型、精神活性物质效应 |
| **主要参数** | 工作记忆错误（重复进入已访问臂）、参考记忆错误（进入无食物臂）、正确选择序列、完成时间 |

---

## 8. Porsolt 强迫游泳实验（Porsolt Forced Swim Test）

| 项目 | 说明 |
|------|------|
| **宫体形状** | 圆柱形透明容器（通常置于更大的水槽中以消除反射） |
| **宫体模板** | 圆形 arena（适配圆柱形容器） |
| **区域模板** | 无特殊区域划分 |
| **范式类型** | 单动物追踪 |
| **适用动物** | 大鼠、小鼠 |
| **实验目的** | 评估抑郁样行为（行为绝望）；测量不动时间反映习得性无助程度 |
| **适用领域** | 抗抑郁药物筛选、抑郁症动物模型、神经递质系统研究（5-HT、NE） |
| **主要参数** | 不动时间（immobility time）、游泳时间、攀爬时间、运动阈值/活动阈值 |

---

## 9. 条件性恐惧实验（Fear Conditioning Test）

| 项目 | 说明 |
|------|------|
| **宫体形状** | 长方形条件性恐惧箱（cubicle），可 1-4 箱并联 |
| **宫体模板** | Ugo Basile Fear Conditioning System（需 FCS 硬件） |
| **区域模板** | 完整箱体为单个区域 |
| **范式类型** | 单动物追踪，事件驱动（声/光/电刺激） |
| **适用动物** | 大鼠、小鼠 |
| **实验目的** | 评估恐惧学习与记忆；通过条件刺激（CS，声/光）与非条件刺激（US，足底电击）的关联测量恐惧反应 |
| **适用领域** | PTSD 研究、焦虑障碍模型、杏仁核功能、记忆巩固/消退研究 |
| **主要参数** | 冻结时间百分比（freezing）、运动活动、事件关联分析（CS-US 间隔、条件反应潜伏期） |

### 关联硬件

- **Noldus Fear Conditioning System (FCS)**：通过串口通信控制声/光/电刺激
- 支持 1-cubicle 和 4-cubicle 配置

## 11. 旋转杆实验（Rotarod / RotaRod Test）

| 项目 | 说明 |
|------|------|
| **宫体形状** | 水平旋转杆（圆形横截面），俯视视角 |
| **宫体模板** | Rotarod |
| **区域模板** | 杆体中心区域（动物在杆上时位于画面中心） |
| **范式类型** | 单动物追踪 |
| **适用动物** | 大鼠、小鼠 |
| **实验目的** | 评估运动协调性和运动学习能力；测量动物在旋转杆上维持平衡的时间 |
| **适用领域** | 帕金森病模型、共济失调研究、运动功能评估、神经肌肉疾病、药物对运动功能的影响 |
| **主要参数** | 掉落/滑落潜伏期、旋转速度（恒定/加速模式） |

## 12. Y 迷宫（Y-maze Test）

| 项目 | 说明 |
|------|------|
| **宫体形状** | Y 形三臂迷宫 |
| **宫体模板** | Y-maze |
| **区域模板** | 三臂区域 + 中央交汇区 |
| **范式类型** | 单动物追踪 |
| **适用动物** | 大鼠、小鼠 |
| **实验目的** | 评估空间工作记忆 |
| **适用领域** | 海马依赖性记忆研究、阿尔茨海默病模型、神经炎症对认知的影响 |
| **主要参数** | 自发交替率（spontaneous alternation %）、各臂进入次数、总运动距离 |

### 变体

- **自发交替（Spontaneous Alternation）**：动物在 Y 迷宫中自由探索，连续进入三个不同臂的序列比例
- **延迟交替（Delayed Alternation）**：在起始臂中限制一段时间后释放

## 13. DanioVision 斑马鱼幼鱼追踪

| 项目 | 说明 |
|------|------|
| **宫体形状** | 多孔板（24/48/96 孔），圆形孔或方形孔 |
| **宫体模板** | DanioVision DVOC 004x（24/48/96 孔）、DVOC 004x/T（带 Tapping Device） |
| **区域模板** | 自适应检测（孔板自动识别） |
| **范式类型** | 多动物追踪（每孔一只幼鱼） |
| **适用动物** | 斑马鱼幼鱼（5-7 dpf） |
| **实验目的** | 高通量药物筛选与毒理学评估；运动轨迹分析、光照反应、声/振动惊跳反应 |
| **适用领域** | 神经药理学、发育毒理学、环境毒理学、遗传学筛选 |

### 特殊功能

- **Tapping Device（/T 模板）**：声/振动惊跳反射（startle response）
- 红外背光 + 白色前光照明系统
- 温控系统（精确控制水温）

## 14. PhenoTyper 家笼观察系统

| 项目 | 说明 |
|------|------|
| **宫体形状** | 标准化透明笼箱（含顶盖集成传感器和照明） |
| **宫体模板** | PhenoTyper / PhenoTyper 2（支持 16 笼配置） |
| **区域模板** | 笼箱全区域 + 自定义子区域（如巢穴、食物区） |
| **范式类型** | 单动物或双动物追踪（需 Social Interaction 模块） |
| **适用动物** | 小鼠（大鼠亦可适配） |
| **实验目的** | 长时间自发行为监测（24h 及以上），昼夜节律分析 |
| **适用领域** | 昼夜节律研究、长期行为监测、社交行为分析、焦虑/抑郁模型 |

## 15. 光遗传学实验（Optogenetics Experiments）

| 项目 | 说明 |
|------|------|
| **宫体形状** | 不限定（可与任意宫体模板配合） |
| **宫体模板** | 自定义 + Prizmatix Pulser/PulserPlus 硬件 |
| **区域模板** | 可基于区域触发刺激（zone-dependent stimulation） |
| **范式类型** | 单/多动物追踪 + 光刺激同步控制 |
| **适用动物** | 小鼠、大鼠 |
| **实验目的** | 通过特定波长的光激活或抑制特定神经元群体，实时操控行为 |
| **适用领域** | 神经环路解析、行为学机制的因果验证、特定脑区功能研究 |

### 特殊功能

- **Prizmatix Pulser 集成**：通过串口精确控制光脉冲序列
- **区域依赖刺激**：动物进入特定区域时自动触发光刺激
- **多通道同步刺激**：支持多个 arena 同时/独立刺激
