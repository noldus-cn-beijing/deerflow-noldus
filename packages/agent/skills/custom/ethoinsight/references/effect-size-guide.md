# 效应量解读

## Cohen's d（两组比较）
| d 值 | 判断 | 行为学意义 |
|------|------|-----------|
| < 0.2 | 可忽略 | 差异无实际意义 |
| 0.2-0.5 | 小效应 | 可能有生物学意义，需更多证据 |
| 0.5-0.8 | 中等效应 | 有明确生物学意义 |
| > 0.8 | 大效应 | 强烈表型差异 |

> **领域注意事项**：Cohen's d 的 0.2/0.5/0.8 是通用规则，但在行为学神经科学中，d = 0.4-0.5 的效应量可能就具有明确的生物学意义。效应量必须结合具体范式的可变性来解读，不要机械套用阈值。

## Eta-squared（多组比较）
| η² 值 | 判断 |
|--------|------|
| < 0.01 | 可忽略 |
| 0.01-0.06 | 小效应 |
| 0.06-0.14 | 中等效应 |
| > 0.14 | 大效应 |

## 补充指标

| 指标 | 适用场景 | 小/中/大 |
|------|---------|---------|
| Hedges' g | t 检验（小样本校正，n < 20 时优先于 Cohen's d） | 0.2 / 0.5 / 0.8 |
| ω² (Omega-squared) | ANOVA（比 η² 更无偏） | 0.01 / 0.06 / 0.14 |
| Partial η² | 多因素 ANOVA（two-way / mixed design） | 0.01 / 0.06 / 0.14 |
| r | Mann-Whitney U 效应量 | 0.1 / 0.3 / 0.5 |
| CLES (Common Language Effect Size) | 任意两组比较，向非统计读者直观解释 | 概率表述，无固定阈值 |

## CLES 解读

CLES 表示"随机从实验组取一只动物、从对照组取一只动物，实验组值更高的概率"。

- CLES = 0.50 → 两组无差异
- CLES = 0.71 → 实验组值更高的概率约 71%（对应 Cohen's d ≈ 0.5）
- CLES = 0.79 → 概率约 79%（对应 Cohen's d ≈ 0.8）

**使用场景**：报告 §4 洞察段首次提及某个指标时，用 CLES 给导师一个直觉理解，配合 Cohen's d 作为正式效应量。

## 表型确认原则

行为表型的确认基于**组间对比**（control vs treatment），不依赖跨实验通用的绝对阈值或常模范围。在此前提下，确认表型需要综合考虑：
1. **方向一致性**: 多个范式或多个范式内指标的信号方向一致（EPM 开臂时间减少 + 旷场中心回避）
2. **效应量**: 统计显著（p < 0.05）配合 Cohen's d > 0.5；仅 p 值不足以下结论
3. **可重复性**: 同一实验室多批次结果一致（若有多批数据）

注意：total distance_moved 或 velocity 差异不作为"表型确认"或"排除运动障碍"的判据——
活动量差异可能反映探索策略不同、造模程度不同等多种原因，不能独立判定病理状态。
