# 效应量解读

## Cohen's d（两组比较）
| d 值 | 判断 | 行为学意义 |
|------|------|-----------|
| < 0.2 | 可忽略 | 差异无实际意义 |
| 0.2-0.5 | 小效应 | 可能有生物学意义，需更多证据 |
| 0.5-0.8 | 中等效应 | 有明确生物学意义 |
| > 0.8 | 大效应 | 强烈表型差异 |

## Eta-squared（多组比较）
| η² 值 | 判断 |
|--------|------|
| < 0.01 | 可忽略 |
| 0.01-0.06 | 小效应 |
| 0.06-0.14 | 中等效应 |
| > 0.14 | 大效应 |

## 补充指标

| 指标 | 适用场景 | 小/中/大 |
|------|---------|---------|
| Hedges' g | t 检验（小样本校正，n < 20 时优先于 Cohen's d） | 0.2 / 0.5 / 0.8 |
| ω² (Omega-squared) | ANOVA（比 η² 更无偏） | 0.01 / 0.06 / 0.14 |
| r | Mann-Whitney U 效应量 | 0.1 / 0.3 / 0.5 |

## 表型确认三层验证

确认焦虑表型需要：
1. **方向一致性**: 多个范式信号方向一致（EPM 开臂减少 + 旷场中心回避）
2. **活动控制**: 总活动距离正常（排除运动障碍）
3. **效应量**: 统计显著（p < 0.05）且 Cohen's d > 0.5
