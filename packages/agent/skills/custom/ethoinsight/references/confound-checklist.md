# 混杂因素检查清单

在解读任何行为指标前，先排除以下混杂因素：

- **运动量差异的处理方式**: 若某 subject 的 distance_moved 或 velocity 与同组其他个体差异较大：
    - 不作为离群值排除依据（运动量低可能是探索策略差异，不是病理异常）
    - 不作为"焦虑指标不可信"的理由
    - 可作为 Discussion 段的混杂因素候选，提示研究员检查是否有独立的健康/造模依据
- **习惯化效应**: 检查时间分段数据，前 1 分钟 vs 后 4 分钟的活动变化
- **Circadian rhythm**: 确认测试时间窗口一致（明暗周期中的相同时段）
- **实验者效应**: 同一批次实验是否由同一人操作
- **装置偏差**: 旷场/EPM 的照度是否一致，迷宫臂是否清洁
- **性别差异**: 雌雄分开分析，不混合
- **年龄/体重**: 确认组间无显著差异
