# 悬尾实验 (Tail Suspension Test (TST))

**slug**：`tail_suspension`  |  **EV19 模板**：调 `identify_ev19_template` 工具获取候选，不要自己列举

## 范式定位

- **一句话定义**：将啮齿动物（主要为小鼠）尾部固定倒悬于空中，通过量化不动行为评估抑郁样行为中的绝望表型
- **测量的心理构念**：抑郁样行为（行为绝望 / behavioral despair）
- **构念维度**：习得性无助样的"放弃挣扎"——动物在无可回避的压迫环境中从挣扎转向被动不动

## 模型概述（为什么有这个实验）

### 临床/研究背景

抗抑郁药研发需要快速、高通量的初筛模型。TST 是经典且快速的评价方法，能区分抗抑郁药、兴奋药、镇静药的药效，且其药效与临床药效显著相关。

### 范式由来

由 Steru 等人 1985 年开发，作为 FST 的替代/补充。核心观察：小鼠尾部固定倒悬后，起初挣扎试图逃脱，经过努力无法摆脱后出现间断性不动，显示"行为绝望"状态。这种状态与抑郁症类似，对绝大多数抗抑郁药敏感。

### 作用与应用

- 抗抑郁药物的初筛（对急性抗抑郁治疗敏感）
- 区分抗抑郁药 / 兴奋药 / 镇静药
- TST 相对 FST 的优势：**无水温/浮力混杂**，更适合不能使用 FST 的情况（如肥胖模型小鼠、较重动物）

## 设计原理（setup 为什么这么设计）

**核心逻辑**：动物在不可逃避的应激（倒悬）中，从主动挣扎转向被动不动。不动时间反映"绝望"程度，与 FST 同构念但物理环境不同（空气 vs 水）。抗抑郁药减少不动时间。

**TST 与 FST 的物理差异及其意义**：
- TST 在空气中，无水温、浮力、阻力混杂 → 干扰因素比 FST 少
- 但 TST 通过尾巴支撑体重 → **不适合较重动物**（大鼠、肥胖模型小鼠会因尾巴承重痛苦），这种情况下应改用 FST

**关键参数的原理**：
- **胶带位置**：距尾尖 1cm（不能靠近尾根，否则限制活动；不能太靠下，影响挣扎）。错误位置直接改变行为
- **悬吊高度 ~30cm**（鼻尖距地面 20-25cm）：确保头倒悬位
- **测试 6 分钟，全程评分**：与 FST 不同，TST 小鼠在实验早期即倾向不动，早期不动本身有信息，**整个 6 分钟全程评分**——不要套用 FST「跳过前段只算后段」的窗口，否则丢失早期不动信号
- **隔间挡板 + 充足宽度**：防止动物间视觉/社交相互作用；动物碰不到箱壁（否则可借力挣扎）
- **背景反差**：背景板与动物毛色强反差（白鼠用黑背景，黑鼠用白背景），便于视频识别

## 指标与判读

### 必算指标

见 `catalog/tst.yaml` 的 `default_metrics`（immobility_time / immobility_latency / immobility_bout_count）。公式以 catalog 为准（SSOT）。

**测量参数差异**：悬尾实验一般采用 **Activity / Activity State** 作为测量参数，而强迫游泳一般采用 **Mobility / Mobility State**，两者不可混用。

### 不分区（无分析区聚合）

TST **不分区**——`catalog/tst.yaml` 无 `zone_concept_params`、无 `in_zone*` requires_columns、无 `anonymous_zone_override`。列对齐链路对 TST **不触发 zone 反问**（`_build_zone_aliases_overrides` 对无 zone_pattern 的范式返回 `{}`）。

**钟摆算法**：不动行为判定使用 pendulum 算法区分钟摆摆动与真实挣扎，完整参数表和算法流程见同目录 `tail_suspension-pendulum-params.md`。data-analyst 在 step 2.8 参数审计时按该文档判据审计。

### 核心指标的第一性原理

- **累计不动时间**：绝望核心代理。不动↑ → 抑郁程度↑。抗抑郁药减少不动时间
- **不动潜伏期**：首次不动时间。缩短 → 更快放弃挣扎
- **挣扎行为时间**：挣扎时间越短，抑郁程度越重

### 解读方向与口径

- 不动时间/次数 **增加** → 抑郁样行为增加；不动潜伏期 **缩短** → 更快绝望
- 抗抑郁药样作用 = 不动时间减少
- **判读只看 control vs treatment 组间显著差异**（P<0.05），不需要单纯比较不动时长绝对值
- **品系基线量化预期（离群归因参照，非判读阈值）**：不同品系基线不动时间差异大（如 C57BL/6 与 ICR 基线可差数量级），看到对照组间基线悬殊先归因品系差异（预期内），而非误判异常个体；品系混杂时组间比较需先按品系分层。此为离群判读的预期参照，**不写成「>Xs 即抑郁」式判读阈值**
- **测量参数不可混用**：TST 用 Activity / Activity State、FST 用 Mobility / Mobility State，二者不可混用；若发现指标参数被混（如用 Mobility 判 TST 不动），导致绝望判读错误，应叫停核对
- **全程评分 vs FST 后段截断**：TST 早期不动本身有信息，按全程评分判读；不要套用 FST 的「只取后几分钟」窗口，否则丢失早期不动信号

## 脱险点 / 数据质量风险

- **爬尾混杂（C57BL/6 特有）**：C57BL/6 等品系常在测试中攀爬自己尾巴，违反 TST「不可逃避」前提（成功爬尾 = 学会逃跑），该样本应排除；防爬管（尾基套透明短管）已被证明不影响药效，未用防爬管且视频可见爬尾的样本排除后需重算 n 与统计功效
- **体重/尾部承重 → 范式适用性**：TST 对较重动物（大鼠、肥胖模型、体重偏大小鼠）有尾痛混杂——尾部承重的疼痛可能被误读为不动。提示改用 FST 或补独立运动活动测试排除运动能力混杂
- **胶带位置错误**：靠近尾根限制活动，太靠下影响挣扎——直接改变不动行为
- **测量参数混淆**：TST 用 Activity/Activity State，FST 用 Mobility/Mobility State，混用导致判读错误
- **品系基线差异**：不同品系空白对照不动时间差异大（如 C57BL/6 vs ICR），跨品系比较需先按品系分层
- **运动活动假阳性**：兴奋剂减少不动但同时增加运动活动，需独立运动测试排查
- **尾部疼痛**：固定方式不当引起疼痛反应，干扰不动判定
- **样本量**：每组 <5 只统计功效不足

## 与其他范式的区分

- **vs FST（强迫游泳）**：都测抑郁样绝望（不动），构念相同但物理刺激不同——TST 空中悬吊（无水温/浮力混杂，但不适合重动物），FST 水中（有水温/浮力/阻力，但适合重动物）。两者结果可相互验证但**不可交叉引用、不可混用判据、测量参数不可混用**
- **vs 焦虑类范式（EPM/OFT/Zero Maze/LDB）**：TST 测抑郁样绝望，焦虑类测焦虑样回避，构念不同
- **TST 与 FST 共用钟摆检测算法**，但参数默认值可能因物理环境（空气 vs 水阻力）不同，各自独立维护
