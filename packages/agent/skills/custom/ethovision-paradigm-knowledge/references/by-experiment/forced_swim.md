# 强迫游泳实验 (Forced Swim Test (FST))

**slug**：`forced_swim`  |  **EV19 模板**：调 `identify_ev19_template` 工具获取候选（如 PorsoltCylinder），不要自己列举

## 范式定位

- **一句话定义**：将啮齿动物置于不可逃避的强迫游泳环境，通过量化不动行为评估抑郁样行为中的绝望表型
- **测量的心理构念**：抑郁样行为（行为绝望 / behavioral despair）
- **构念维度**：习得性无助样的"放弃挣扎"——动物从主动逃逸转向被动不动

## 模型概述（为什么有这个实验）

### 临床/研究背景

抑郁症是重大疾病负担，抗抑郁药研发需要高通量、可靠的初筛模型。FST 是评估抗抑郁样行为最广泛使用的动物模型之一，因其高通量、易用、实验室间可靠性和特异性。

### 范式由来

由 Porsolt 等人开发。核心观察：大鼠暴露于水中后，最初表现强烈逃逸导向行为（游泳、攀爬），随后停止挣扎、表现出被动不动。这种从主动到被动的转变被解释为"行为绝望"——一种类似抑郁的表型。**抗抑郁药样作用 = 减少被动不动行为**。

### 作用与应用

- 筛选抗抑郁药（FST 对急性抗抑郁治疗敏感）
- 区分药物作用的神经递质机制（修改版 FST，见下）
- 抑郁样表型的模型验证

## 设计原理（setup 为什么这么设计）

**核心逻辑**：动物在不可逃避的应激（强迫游泳）中，从主动逃逸转向被动不动。不动时间反映"绝望"程度。抗抑郁药减少不动时间。

**原始 FST vs 修改版 FST 的关键区别**（这是 FST 最重要的方法论演进）：
- **原始 FST**（Porsolt）：水深 15-18cm，累积计时测不动时间。**问题：对 SSRIs（选择性 5-HT 再摄取抑制剂）检测不可靠**
- **修改版 FST**（Luki 等）：水深增至 30cm（大鼠无法用尾巴在缸底稳定自己 → 减少假性不动）+ **时间采样技术**（每 5s 评估一次主要行为）。关键改进：把逃逸行为分解为**攀爬**（前爪垂直移动）和**游泳**（水平移动）两类
- **神经递质机制的分离**：儿茶酚胺能药物选择性增加**攀爬**；5-HT 能药物选择性增加**游泳**。这使得修改版 FST 能区分药物作用的主要神经递质系统

**关键参数的原理**：
- **水温 23-25°C**：过高（如 30°C）大鼠仅漂浮；过低（15-20°C）导致体温过低且动物更活跃（混杂）
- **水深 30cm（修改版）**：确保动物无法用尾巴支撑在缸底（否则产生假性不动）
- **缸径 ≥20cm 但不能太宽**：太宽大鼠可自由游泳，行为模式改变
- **大鼠需预游泳**（测试前 24h）：确保测试当天快速采取不动姿势，便于观察药效。**小鼠不需要预游泳**（单次暴露即有稳定不动基线）

## 指标与判读

### 必算指标

见 `catalog/fst.yaml` 的 `default_metrics`（immobility_time / immobility_latency / immobility_bout_count）。公式以 catalog 为准（SSOT）。

**钟摆算法**：不动行为检测使用自相关周期性分析（区分真实不动 vs 停止挣扎后的钟摆摆动），参数判据见同目录 `forced_swim-pendulum-params.md`。data-analyst 在 step 2.8 参数审计时，对 FST pendulum 参数按该文档判据审计。

### 不分区（无分析区聚合）

FST **不分区**——`catalog/fst.yaml` 无 `zone_concept_params`、无 `in_zone*` requires_columns、无 `anonymous_zone_override`。列对齐链路对 FST **不触发 zone 反问**（`_build_zone_aliases_overrides` 对无 zone_pattern 的范式返回 `{}`）。即使数据带 zone 味道的列也不应被激活。

### 核心指标的第一性原理

- **累计不动时间**：绝望核心代理。不动↑ → 抑郁样行为↑。抗抑郁药减少不动时间
- **不动潜伏期**：首次不动的时间。潜伏期缩短 → 更快放弃挣扎
- **不动次数**：不动发作的次数
- **攀爬 vs 游泳**（修改版 FST）：区分神经递质机制——攀爬↑提示儿茶酚胺能，游泳↑提示 5-HT 能

### 解读方向与口径

- 不动时间/次数 **增加** → 抑郁样行为增加；不动潜伏期 **缩短** → 更快绝望
- 抗抑郁药样作用 = 不动时间减少
- **判读只看 control vs treatment 组间显著差异**，不用绝对阈值
- **关键混杂排查**：减少不动时间的治疗，必须排除"兴奋剂样增加运动活动"的假阳性——需配合独立的运动活动测试（如旷场）
- **SSRI × FST 版本适配**：按 FST 版本判读。若干预为 5-HT 能（SSRI 类）但数据来自原始 FST（不分攀爬/游泳），提示其检测可靠性有限，报告应注明；修改版 FST（攀爬/游泳分解）才能区分神经递质机制
- **攀爬/游泳 → 机制映射 + 反证检验**（修改版 FST，组间比较）：仅攀爬↑提示儿茶酚胺能、仅游泳↑提示 5-HT 能、二者皆↑提示多靶点。若数据模式与声称机制矛盾（如只增游泳却称儿茶酚胺能作用），标红待复查

## 脱险点 / 数据质量风险

- **运动活动假阳性/假阴性 + 测试顺序**：兴奋剂会减少不动时间但同时也增加一般运动活动 → FST 假阳性；镇静剂/运动损害化合物导致假阴性。不动下降（尤其疑似兴奋剂样作用）须配独立运动活动测试排除假阳性；若同一动物复用，运动测试应在 FST 之前——FST 有长期行为影响，会污染后续运动基线。报告中无独立运动测试时，不动下降不可单独判为抗抑郁
- **水温混杂**：水温不当直接影响不动时间（过高漂浮、过低体温过低）
- **水深不足**：<30cm 时大鼠用尾巴支撑缸底产生假性不动
- **品系基线量化预期（离群归因参照，非判读阈值）**：跨品系基线不动可差数量级（小鼠品系间差异可达 10 倍；Wistar-Kyoto 大鼠比 SD 更多不动且攀爬↓）。看到对照组间基线悬殊，先归因品系差异（预期内），而非误判为异常个体或实验故障；品系混杂时组间比较本身需先按品系分层。此为离群判读的预期参照，**不写成「>Xs 即抑郁」式判读阈值**
- **季节性混杂**：同实验室跨批/长期数据汇总时，注意季节可能混杂基线不动（冬↑夏↓）；跨季节对比应分季节看效应，避免季节主效应掩盖干预效应
- **预游泳要求**：大鼠需 24h 前预游泳，小鼠不需要——物种差异，不能混淆流程
- **SSRIs 检测**：原始 FST 对 SSRIs 不可靠，需用修改版 FST（区分攀爬/游泳）
- **样本量**：大鼠每组 n=8-12

## 与其他范式的区分

- **vs TST（悬尾）**：都测抑郁样绝望（不动），但物理刺激不同——FST 在水中（有水温、浮力、阻力混杂），TST 在空中悬吊。两者结果可相互验证但**不可交叉引用、不可混用判据**
- **vs 焦虑类范式（EPM/OFT/Zero Maze/LDB）**：FST 测抑郁样绝望，焦虑类测焦虑样回避，构念不同
- **FST 与 TST 共用钟摆检测算法**（自相关周期性分析），但参数默认值可能因物理环境（水阻力 vs 空气）不同，各自独立维护
