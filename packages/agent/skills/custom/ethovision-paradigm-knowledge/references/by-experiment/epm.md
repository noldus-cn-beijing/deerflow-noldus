# 高架十字迷宫 (Elevated Plus Maze (EPM))

**slug**：`epm`  |  **EV19 模板**：调 `identify_ev19_template` 工具获取候选（如 PlusMaze-AllZones），不要自己列举

## 范式定位

- **一句话定义**：利用啮齿动物对开放高处的天然回避倾向，通过量化开放臂探索行为评估焦虑样行为
- **测量的心理构念**：焦虑（anxiety）
- **构念维度**：接近-回避冲突（approach-avoidance conflict）——动物探索新异环境的倾向 vs 对高悬开阔空间的先天恐惧之间的矛盾

## 模型概述（为什么有这个实验）

### 临床/研究背景

焦虑症是全球重大疾病负担，但病理生理学进展有限，核心难点是病因异质性。动物行为模型不模拟特定疾病，而是模拟特定症状（如焦虑样回避），用于筛选抗焦虑药物、研究焦虑的神经机制。EPM 是测量状态性、非条件焦虑的国际公认经典模型。

### 范式由来

源于 Montgomery 1955 的"Y"形高架装置——他观察到大鼠探索高架开/闭臂时表现出探索与厌恶的接近-回避冲突。Handley 和 Mithani 改编为现在的十字形（两条开臂 + 两条闭臂）。

### 作用与应用

- 筛选/评价药物的抗焦虑或致焦虑作用（对经典抗焦虑药敏感，且反应有时程效应）
- 研究焦虑的脑区（边缘系统、海马、杏仁核）和神经机制（GABA、5-HT、HPA 轴）
- 焦虑相关表型的模型验证

## 设计原理（setup 为什么这么设计）

**核心逻辑**：啮齿动物有探索新异环境的习性，又厌恶强光和开阔地。当被迫处于高架开臂，动物表现出恐惧反应（freezing、排便排尿、血浆糖皮质激素升高）。这种厌恶的根源是**趋触性（thigmotaxis）**——动物靠近垂直表面以避免被捕食者发现的防御本能。开臂无垂直壁 → 厌恶 → 花更多时间在闭臂。开臂探索比例因此成为焦虑的可量化代理指标。

EPM 是**非条件反射**焦虑模型（区别于依赖电击/剥夺的条件性模型），优势是**无需训练、单次测试、结果重复性好**。

**关键参数的原理**：
- **测试 5 分钟**：基于 Montgomery 研究，大鼠前 5 分钟对开臂回避反应最强，是信号最丰富的窗口
- **起始位置统一**（中央区、面向开臂）：起始位置/方向影响结果，必须每只一致
- **四臂光照一致**：避免光照混杂（强光本身增加焦虑）
- **离地高度 50-70cm（大鼠）/40-55cm（小鼠）**：足够高以唤起回避，但不能高到动物受伤

## 指标与判读

### 必算指标

见 `catalog/epm.yaml` 的 `default_metrics`（open_arm_time_ratio / open_arm_entry_ratio / open_arm_time / open_arm_entry_count / total_entry_count）。指标公式以 catalog 为准（SSOT），本文件不重复。

### 自定义分析区列的聚合语义（HITL 列对齐时消费）

用户在 EthoVision「按分析区输出」自定义的开放臂/封闭臂列，可能不止一列（两条开放臂分别导出）。本系统按**同一分析区群组内 OR 合并**聚合：

- **两条开放臂（open arm A + open arm B）→ OR 合并**成单一"开放臂"概念（`open_arm_zones` 收两列，逐帧 `max`＝动物在任一臂即计在开放臂）；两条封闭臂同理。中心区一般不导出，不参与。
- **同一群组内 OR 安全**（互斥，工具强制）：两条开放臂坐标不可能同帧为 1，OR 不会双重计数。
- ⚠️ **累积分析区陷阱**（跨群组堆叠）：若用户同时导出了分量列（open arm A / open arm B）**和** EV 内已聚合的累积列（open arm all），三者都别名成"开放臂"时——因 `all = A ∪ B` 支配关系，逐帧 `max(A, B, all) == all`，OR 合并**数值仍正确**（不会被双重计数放大）。但**绝不能改成逐列相加**（会把占比算成 >1 的物理不可能值）。列对齐 HITL 时若发现同时存在分量列 + 累积列，优先让用户确认哪列是"累积/整合区"以择一，避免依赖支配关系侥幸正确。

聚合规则权威来源：`docs/review-packages/2026-06-09-feedbacks/自定义数据列识别对齐聚合.md`（同事方法论）。

### 核心指标的第一性原理

- **开臂停留时间百分比 / 开臂进入次数百分比**：直接反映接近-回避冲突的偏向。焦虑↑ → 开臂探索↓
- **总入臂次数**：反映总体活动水平，**用于排除运动抑制混杂**——若总入臂过低，开臂指标下降可能是运动抑制而非焦虑
- **风险评估指标**（次要指标，对低剂量药物可能比开臂时间更敏感）：
  - **开臂边缘探出 / 低头探索（head-dipping）**：EPM 有开臂（无外墙边缘），动物可从平台边缘探头向下张望，故 head-dipping 在 EPM 适用（**带墙范式如 OFT/LDB 不存在此行为**）。通过**分析区**测量——EPM 导出含 `nose_over_edge_open_arms` 等"鼻尖越过开臂边缘"的分析区列，以 catalog 实际登记的指标为准（SSOT）。
  - **伸展注意（SAP）/ 身体伸长（elongation）**：实践中**很少使用**，非本系统常用指标，**默认不算**、报告不提；仅当用户明确要求时才基于 `elongation` 列计算。
  - **后腿直立（rearing）**：⚠️ **本系统不支持**。rearing 本质是"速度不变（原地）+ 身长变短 + 投影面积变小"，理论上可从 elongation 等推算，但 **EthoVision 精细行为分类器信效度不佳**，且 rearing 在焦虑/旷场范式里是**非必须指标**（只在专门测精细行为的范式才需要，那类实验不在支持范围）。本系统不产出，**报告中不得编造该指标的数值**。

### 解读方向与口径

- 开臂时间/进入百分比 **降低** → 焦虑样行为增加；**升高** → 抗焦虑作用
- **判读只看 control vs treatment 组间是否有显著差异**，不用"开臂时间正常范围 X-Y%"这类绝对阈值（绝对阈值只能作批次质检参考，不作判读依据）
- 必须同时报告**总入臂次数**以排除运动抑制——单纯开臂↓而总入臂也↓，结论应是运动抑制而非焦虑

## 脱险点 / 数据质量风险

- **一次试验耐受现象**：二次测试开臂活动显著减少，故 EPM **通常只测一次**。若研究者坚持重测，提示其满足「间隔 ≥3 周且更换房间」才能避免一次试验耐受（开臂基线不下降）；否则二次测试开臂下降可能是耐受而非干预效应。需重复测量的研究优先建议改用 Zero Maze（无耐受）
- **预暴露混杂**：测试前对新环境的预暴露会**增加**开臂活动与探索。若各组测试前的环境处理不一致（如仅处理组做了新环境预暴露），开臂指标升高可能源于预暴露而非干预效应——判读前确认各组前处理一致
- **运动抑制混杂**：总入臂次数过低（运动抑制）时，开臂指标下降不等于焦虑增加，需标注警告
- **轨迹断层/异常静止的剔除溯源**：轨迹出现断层或异常静止时，区分三种来源——① 从开臂边缘掉落（开臂后轨迹消失，该样本数据应剔除）② 外部噪音致冻结（符合排除标准、应剔除）③ 真实低探索个体。剔除决定要可追溯，报告须透明注明剔除 n / 总 n
- **冻结反应干扰**：开臂上遇噪音/人活动会引发长时间 freezing，占据测试时长——噪音干扰的动物数据应剔除
- **基线差异**：年龄↑、性别、发情周期都会改变开臂基线。设计时需控制（许多研究只用雄性避免发情周期干扰）
- **对比度问题**：动物排尿导致反光 → 视频跟踪失准，需提前调试对比度
- **样本量**：每组 <5 只统计功效不足

## 与其他范式的区分

- **vs Zero Maze（O迷宫）**：都测焦虑接近-回避冲突，结构不同——EPM 十字形有中央区（动物在此花 20-30% 时间，是解释的模糊地带），Zero Maze 环形无中央区、且无耐受现象。**需要重复测量选 Zero Maze，单次测量两者均可**
- **vs 旷场（OFT）**：OFT 不依赖高度，焦虑通过中心区回避体现；EPM 通过开放高处回避体现。OFT 还能同时评估自主活动性
- **vs FST/TST**：EPM 测焦虑（回避），FST/TST 测抑郁样绝望（不动），构念不同不可混用
