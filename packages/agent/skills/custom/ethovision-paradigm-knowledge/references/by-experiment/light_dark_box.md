# 明暗箱 (Light-Dark Box)

**slug**：`light_dark_box`  |  **EV19 模板**：调 `identify_ev19_template` 工具获取候选（明暗箱无专用模板时可能是 Subdivided 或自定义，需同事确认）

## 范式定位

- **一句话定义**：利用啮齿动物对明亮环境的天然回避倾向，通过量化明暗两区间的探索行为评估焦虑样行为水平
- **测量的心理构念**：焦虑（anxiety）
- **构念维度**：基于光照厌恶的探索-回避冲突（明箱=厌恶/威胁区，暗箱=安全区）

## 模型概述（为什么有这个实验）

### 临床/研究背景

焦虑测试需要多样、互补的模型。LDB 基于啮齿动物**天生偏好暗处、回避明亮开放空间**的本能，提供一个比 EPM/OFT 更简单的焦虑测量（无高度应激）。主要在小鼠中开发和使用。

### 范式由来

基于动物明暗偏好本能的经典焦虑模型。Hascoët 和 Bourin 等人系统化了测试程序。

### 作用与应用

- 筛选抗焦虑/致焦虑药物（对苯二氮卓类敏感）
- 焦虑神经生物学基础研究
- 与 EPM/OFT 互补——三者测焦虑但应激源不同（光照 vs 高度 vs 开阔），可相互验证

## 设计原理（setup 为什么这么设计）

**核心逻辑**：啮齿动物天生回避明亮环境（明亮 = 暴露于捕食者的威胁）。LDB 提供明暗两个隔室，动物在探索本能（探索明箱）与回避本能（回避光亮）间冲突。焦虑↑ → 明箱探索↓。明箱停留时间是与药物剂量效应最一致的测量。

**关键参数的原理**：
- **明暗隔室比例**：常见 1/3 暗箱 + 2/3 明箱，或 1/2 + 1/2。比例影响基线但不改变构念
- **连接门 7×7cm**：保证小鼠自由通过但不过大（过大会减弱两区区分）
- **光照强度差异是核心应激源**：暗箱 ≤5 lux，明箱光强度因品系而异（需预实验选适中基线）。**光强度是探索行为的强大环境因素**——高强度（330lux）减少探索且效果不超 15min；中等（65lux）促进 10min 内持续探索
- **顶部覆盖**：限制外界气味干扰；明箱透明顶、暗箱不透明盖
- **5 分钟测试**：标准时长
- **不可重复测试、不可提前适应装置**：测试依赖环境的新奇性。苯二氮卓类第一次测试有抗焦虑作用，第二次完全消失——故只能组间比较，不可自身对照

## 指标与判读

### 必算指标

见 `catalog/ldb.yaml` 的 `default_metrics`（明箱时间百分比 / 穿梭次数 / 潜伏期）。公式以 catalog 为准（SSOT）。

### 核心指标的第一性原理

- **明箱停留时间（百分比）**：与药物剂量效应**最一致**的测量，焦虑核心代理
- **穿梭次数**：反映探索与回避的平衡。**但易受运动活动混杂**——若药物增加一般活动，穿梭次数假性升高。穿梭↑但不伴随自发运动↑ 才是抗焦虑
- **首次进入明箱潜伏期**：反映回避强度
- **伸展注意（SAP）/ 身体伸长（elongation）**：实践中**很少使用**，非本系统常用指标，**默认不算**、报告不提；仅当用户明确要求时才基于 `elongation` 列计算。
- ⚠️ **以下指标本系统不产出**，**报告中不得编造其数值**；如用户问及，按说明回应：
  - **向外探出暗箱（lean-out / peek-out）**：LDB 是封闭箱（无开放边缘），且明暗区判定**以身体中心点为标准、不测量 nose-in-zone**，故无"鼻尖探出/低头"指标。
  - **后腿直立（rearing）**：本系统不支持——精细行为分类器信效度不佳，且在焦虑范式里非必须（详见 EPM/OFT 说明）。

### 解读方向与口径

- 明箱停留时间/百分比 **降低** → 焦虑样行为增加；**升高** → 抗焦虑作用
- **穿梭次数的双重性**：穿梭次数既是焦虑指标又是运动量指标。仅当穿梭↑（组间）**不伴随自发运动↑** 才提示抗焦虑；若一般活动同时↑，穿梭↑可能是运动假阳性而非抗焦虑
- **潜伏期的起始位置二重性**：潜伏期含义随起始位置反转——起始=明箱时，进暗箱潜伏期反映避明强度、返明箱潜伏期反映回避恢复；起始=暗箱时，进明箱潜伏期反映探索动力。报告潜伏期须注明起始位置
- **判读只看 control vs treatment 组间显著差异**，不用绝对阈值
- **关键**：必须综合多指标判读，不能只看单一变量（单一变量易受非特异性效应污染，见脱险点）

## 脱险点 / 数据质量风险

- **视觉能力混杂**：失明小鼠（如 C3H 品系视网膜变性）无法区分明暗 → 数据无法在焦虑框架解释。必须确认处理因素不改变视觉能力
- **镇静假阳性**：高剂量苯二氮卓类诱导镇静，明箱进入减少可能是镇静而非焦虑——必须结合多指标
- **运动活动混杂穿梭次数**：兴奋性药物增加穿梭但非抗焦虑。穿梭↑不伴随自发运动↑ 才是抗焦虑
- **品系差异巨大**：9 种品系明暗箱行为差异极大（A/J/BALB 高焦虑，DBA/C57 低焦虑）。C57BL/6J 和 Swiss-NIH 是首选品系
- **年龄/体重混杂**：4 周龄 Swiss 小鼠对照值最佳（暗箱停留 58%）；8 周龄总活动增加、穿梭增加。解释需谨慎
- **光强度-品系交互作用**：光强度效果与品系有交互，需对特定品系做光强度预实验
- **起始位置影响基线**：放明箱中心 vs 暗箱角落，基线不同——必须每只一致
- **测试时间/噪音/笼子组成混杂**：需半随机测试顺序（平衡昼夜节律、噪音、笼子组成）
- **不可重复测试**：第二次测试抗焦虑作用消失
- **明暗区光照差异不足**：动物无法区分两区，数据失效

## 与其他范式的区分

- **vs EPM / Zero Maze**：LDB 不依赖高度，通过**光照强度差异**产生回避；EPM/Zero Maze 通过开放高处回避。应激源不同，可互补验证
- **vs OFT**：LDB 有明确物理隔断（明暗两区），OFT 为单一开放空间。LDB 应激源是光照，OFT 是开阔
- **vs FST/TST**：LDB 测焦虑（光照回避），FST/TST 测抑郁样绝望，构念不同
