# 旷场实验 (Open Field Test (OFT))

**slug**：`open_field`  |  **EV19 模板**：调 `identify_ev19_template` 工具获取候选（如 OpenFieldRectangle），不要自己列举

## 范式定位

- **一句话定义**：利用啮齿动物对开阔环境的天然回避倾向，通过量化中心区与周边区探索行为评估焦虑样行为水平和自主活动性
- **测量的心理构念**：焦虑（anxiety）+ 自主活动性（locomotor activity）—— **OFT 是少数能同时评估两者的范式**
- **构念维度**：趋触性驱动的中心区回避

## 模型概述（为什么有这个实验）

### 临床/研究背景

焦虑样障碍是全球前十大致残原因之一，女性患病率约为男性 1.75 倍。理解焦虑机制需要可靠的动物模型。OFT 是应用最广泛的焦虑测试之一，因其自动化程度高、数据丰富、可行性高。

### 范式由来

由 Calvin Hall 于 1934 年设计发明，最初用于研究大鼠情绪变化（记录排便评分），后推广到小鼠及其他动物。现代版用自动视频跟踪分析中央区活动作为焦虑指标。

### 作用与应用

- 评价药物/环境对焦虑样行为和自主活动的影响
- 检验造模是否成功（慢性疲劳、PTSD、失眠等模型）
- 焦虑神经生物学基础研究、抗焦虑化合物筛选
- **重要提醒**：OFT 测焦虑的结果不应过度解读，应结合其他焦虑测试（EPM、明暗箱）综合分析

## 设计原理（setup 为什么这么设计）

**核心逻辑**：啮齿动物面对新异环境存在两种对立倾向——好奇心驱使探索 vs 恐惧回避明亮开阔场所。其活动具有**趋触性（thigmotaxis）**：探索开放空间时本能"贴墙"。利用探索动机与恐惧回避的矛盾冲突，既能评估焦虑（中心区回避），又能评估一般活动（总运动距离）。这是 OFT 区别于 EPM 的核心特点——**双重用途**。

**关键参数的原理**：
- **正方形箱体**（啮齿标准；圆形消除角落聚集，适合鱼类/昆虫）
- **中心/周边划分**：通常箱底划 25 格，侧壁格=周边区，其余=中心区
- **测试 15 分钟**：前 15 分钟更多体现探索行为，15 分钟后主要反映一般活动能力
- **运动阈值 3.0 cm/s**：区分运动/静止的阈值，可事后调整但需人工核对
- **起始位置统一**（中心区）：每只位置方向一致

## 指标与判读

### 必算指标

见 `catalog/oft.yaml` 的 `default_metrics`（center_time_ratio / center_distance_ratio / center_entry_count / center_time / center_distance + 总运动距离相关）。公式以 catalog 为准（SSOT）。

### 核心指标的第一性原理

- **中心区时间百分比 / 距离百分比 / 进入次数**：焦虑核心代理。焦虑↑ → 趋触性增强 → 贴墙（周边区）→ 中心区指标↓
- **总运动距离**：自主活动性代理，**用于排除运动抑制混杂**
- **直立（rearing）**：探索行为指标。关键细分——
  - **无支撑直立**（短暂站立不靠墙）：与中心区活动相似，**焦虑敏感指标**（与排便负相关）
  - **支撑直立**（靠墙站立）：与距离/活动相关，反映运动能力而非情绪

### 解读方向与口径

- 中心区指标 **降低** → 焦虑样行为增加；**升高** → 抗焦虑作用
- 总运动距离 **降低** → 自主活动下降 / 运动抑制 / 抑郁样表现
- **判读只看 control vs treatment 组间显著差异**，不用"中心时间正常范围 X%"绝对阈值
- 必须同时报告**总运动距离**——单纯中心区↓而总距离也↓，结论应是运动抑制而非焦虑
- 直立行为遵循**倒 U 型曲线**：不确定时（探索动力≈危险感知）直立最多；极度危险或极度安全（适应后）都减少。解读取决于基线在曲线的位置

## 脱险点 / 数据质量风险

- **运动抑制混杂**：总运动距离过低（如 <1000cm）时，中心区指标下降可能是运动抑制而非焦虑
- **中心区定义不一致**：不同实验室中心区划分方案不同 → 结果不可比，需确认 zone 方案
- **直立行为的复杂性**：不区分支撑/无支撑直立会丢失情绪信息；直立有显著性别差异
- **测试时长影响解读**：前 15min 探索 vs 后段一般活动，分析时需明确
- **品系差异**：C57BL/6 是焦虑研究"黄金品系"（探索少、自发活动低、受性别/昼夜影响小）
- **序贯测试顺序**：一批动物做多个实验时建议**先做 OFT**（对其他实验影响小）；但游泳类（水迷宫）会引起压力影响 OFT 指标
- **样本量**：大鼠 ≥8只/组，小鼠 10-12只/组

## 与其他范式的区分

- **vs EPM / Zero Maze**：OFT 不依赖高度，焦虑通过中心区回避体现（趋触性）；EPM/Zero Maze 通过开放高处回避体现。OFT 还能同时评估自主活动性，这是它的独特优势
- **圆形 vs 矩形旷场**：圆形消除角落聚集，适合鱼类/昆虫；矩形是啮齿标准
- **vs FST/TST**：OFT 测焦虑+活动性，FST/TST 测抑郁样绝望，构念不同
