# 零迷宫 (Zero Maze / Elevated Zero Maze)

**slug**：`zero_maze`  |  **EV19 模板**：调 `identify_ev19_template` 工具获取候选（如 ZeroMaze），不要自己列举

## 范式定位

- **一句话定义**：利用啮齿动物对开放高处的天然回避倾向，通过环形迷宫中开放区与封闭区的探索行为评估焦虑样行为水平
- **测量的心理构念**：焦虑（anxiety）
- **构念维度**：接近-回避冲突（approach-avoidance conflict）—— Zero Maze 是 EPM 的改良版，针对 EPM 中央区模糊的设计缺陷

## 模型概述（为什么有这个实验）

### 临床/研究背景

焦虑模型需要可靠、敏感、可重复测量。EPM 虽经典但有方法论缺陷（中央区模糊），Zero Maze 针对这些缺陷改良，尤其适合需要重复测量的纵向研究。

### 范式由来

由 Shepherd 等人 1994 年首次描述，用苯二氮卓类药物验证。是 EPM 的改良版，设计目标：① 增加对药物作用的敏感性；② 促进对药物作用的解释（消除中央区模糊）；③ 允许不间断探索。

### 作用与应用

- 评价药物的抗焦虑/致焦虑作用（对苯二氮卓类敏感，且详细行为分析增加敏感性）
- 研究焦虑的神经机制（脑区、神经递质）
- **特别适合需要重复测量的纵向研究**（无耐受现象——这是相对 EPM 的核心优势）

## 设计原理（setup 为什么这么设计）

**核心逻辑**：与 EPM 相同——啮齿动物探索新异环境的倾向 vs 对高悬开放空间的先天恐惧，形成接近-回避冲突。焦虑水平决定冲突偏向。

**Zero Maze 相对 EPM 的设计改进——每个改进都有明确目的**：
- **环形无中央区**：EPM 十字形有中央区，小鼠在此花 20-30% 时间，这部分无法归入开/闭臂，**损害对开/闭臂比较的解释**。Zero Maze 完全去除中央区 → 开放/封闭时间二分干净
- **环形=不间断探索**：EPM 中央区是"决策节点"动物会停下犹豫；Zero Maze 环形让动物**持续处于开放或封闭**，不存在"停在哪都不算"的状态
- **更大活动区域**：环形比十字提供更大空间 → 增加接近-回避冲突发生概率 → 信号更丰富。C57BL/6J 数据显示 Zero Maze 开放区停留时间是 EPM 的**两倍**（鼓励更深入探索）

**关键参数的原理**：
- **5 分钟测试**：基于 Montgomery 研究，大鼠高架开臂前 5 分钟回避反应最强（同 EPM）
- **开放象限仅 1cm 高边沿**：作触觉指导，非真正屏障
- **光照 40-60 lux（开放/封闭一致）**：消除光照混杂（光照差异本身会增加焦虑）
- **离地高度**：大鼠 50-70cm，小鼠 40-50cm
- **起始位置无统一标准**（可放封闭/开放/交界），但**关键是每只一致**

## 指标与判读

### 必算指标

见 `catalog/zero_maze.yaml` 的 `default_metrics`（开放区时间百分比 / 开放区滞留时长 / 开放区移动距离 / 犹疑次数 / 总移动距离）。公式以 catalog 为准（SSOT）。

### 核心指标的第一性原理

**结果型指标**（冲突的最终偏向）：
- **开放象限停留时间百分比 / 进入次数百分比**：焦虑核心代理
- **总移动距离**：总体活动水平，**用于排除运动抑制混杂**

**过程型/风险评估指标**（冲突的决策瞬间）——**这才是 Zero Maze 的洞察所在**：
- **伸展注意姿势（stretch-attend）/ 犹疑次数**：动物在封闭区拉长身体、鼻子越过开-闭分界线又缩回。**捕捉接近-回避冲突的矛盾状态本身**——这是冲突最直接的行为外显
- **低头向下探索（head-dipping）**：在开放/封闭象限末端从平台边缘向下探，风险评估行为

**核心洞察——为什么伸展注意是最敏感的情绪指标**：
> 在动物从封闭象限接近开放象限时，伸展注意频率提供**最敏感的情绪变化指数**。关键实证：在比显著影响开放象限活动**更低的剂量**下，抗焦虑/致焦虑药就对低头探索和伸展注意有明显影响。

- 开放区停留时间 = 冲突的**结果**（已决策）→ 药物需达一定剂量才改变
- 伸展注意 = 冲突的**过程**（正在矛盾）→ 对药物**更敏感**，低剂量即可检测

### 解读方向与口径

- 开放区时间/进入百分比 **降低** → 焦虑样行为增加；伸展注意**减少** → 焦虑减少（与开放区时间一致方向）
- **判读只看 control vs treatment 组间显著差异**，不用绝对阈值
- 必须同时报告**总移动距离**排除运动抑制
- 应报告**风险评估指标**（伸展注意/低头探索），否则可能漏掉低剂量药物效应

## 脱险点 / 数据质量风险

- **运动抑制混杂**：总移动距离过低时，开放区指标下降可能是运动抑制而非焦虑
- **犹疑次数的主观性**：犹疑次数（伸展注意）判定依赖标准，不同实验室可能不一致——需明确定义（鼻子越过开-闭分界线又缩回）
- **光照混杂**：开放/封闭光照必须一致（40-60lux），否则光照差异混杂
- **起始位置影响基线**：起始位置无统一标准但必须每只一致
- **犹疑次数的倒 U 特性**：需注意极端焦虑或极端安全都会减少探索行为
- **样本量**：每组 <5 只统计功效不足

## 与其他范式的区分

- **vs EPM**：**构念相同**（都测焦虑接近-回避冲突），但 Zero Maze 有三大方法论优势——① 环形无中央区（消除模糊）；② 无一次试验耐受现象（可重复测量，EPM 有耐受）；③ 鼓励更深入探索。**需要重复测量/纵向追踪选 Zero Maze，单次测量两者均可**
- **vs OFT**：Zero Maze 依赖高度差异产生回避，OFT 依赖空间开阔度产生回避
- **vs FST/TST**：Zero Maze 测焦虑（回避），FST/TST 测抑郁样绝望（不动），构念不同
