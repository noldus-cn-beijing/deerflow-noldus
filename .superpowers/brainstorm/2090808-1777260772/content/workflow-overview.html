Lead Agent 新工作流：手动模式（飞轮期）

红色标注 = Human-in-the-loop 确认点 | 蓝色 = Subagent | 绿色 = 最终选项

Phase 1 — 实验识别
1a Lead Agent 询问："您需要做什么？"
选项：端到端数据分析 / 知识问答 / 其他
→
1b Lead Agent 展示 16 种实验模板
用户选择 → 数据一致性 safety check
→
🔴 Gate 1
用户确认实验类型
Phase 2 — 实验设计确认
2a Lead Agent 追问组别信息
对照组 vs 实验组？每组几个动物？
→
🔴 Gate 2
用户确认组别设计 + 分析目标
→
🔵 code-executor
5 步分析流水线（按范式模板）
Phase 3 — 初步结论审查
🔵 data-analyst
统计审核 + 异常检测 + 洞察发现
→
🔴 Gate 3
展示初步结论 → 用户标记深入方向
Phase 4 — 最终产出
🔵 report-writer
APA 科学报告（如用户选择）
→
✅ 完成
可选择报告 / 追问 / 新分析

自动模式（post-training 后）

选实验 + 传数据
→
全自动 pipeline
无中断，端到端
→
最终结果 + 可深入方向
